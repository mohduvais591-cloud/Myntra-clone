# Myntra Clone

A modern and responsive fashion e-commerce web application inspired by Myntra. This project replicates the core functionalities of an online shopping platform, providing users with an intuitive and seamless shopping experience.

## 🚀 Features

* Responsive and user-friendly design
* Product listing and catalog display
* Product search functionality
* Category-based filtering
* Shopping cart management
* Dynamic product details page
* Modern UI/UX design
* Mobile-friendly layout

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Bootstrap
